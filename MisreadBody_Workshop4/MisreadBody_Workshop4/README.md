## Instructions
1. Open the TouchDesigner project file: `MisreadBody.toe`
2. Make sure a webcam is connected and available
3. If needed, check the camera input in the Video Device In TOP and select the correct device
4. Run the project
5. Stand in front of the camera and move your body or arms
6. The system detects movement and translates it into fading trails, scanned edges, color-mapped residues, and distorted visual traces
7. If the output is too bright or too weak, adjust the final output level nodes if necessary
8. For best results, use a relatively clean background and stable lighting conditions

## Blurb
Misread Body is a camera-based prototype developed in TouchDesigner. It explores the gap between being seen by a machine and being understood by it. Instead of presenting the body directly, the system detects motion and transforms it into unstable traces, delayed residues, and distorted contours. The work suggests that machine vision does not fully understand the body, but produces a partial and computationally reduced version of it.

## References
1.	Gunderson, M., Rettberg, J.W., Solberg, R. and others (2024) ‘Machine vision situations: Tracing distributed agency’, Convergence. 10.12688/openreseurope.16112.2
This reference treats machine vision as something that actively enters a situation and changes what happens, rather than as a neutral tool. The article defines machine vision as the registration, analysis, and representation of visual data by machines and algorithms, and argues that artworks are a strong site for analysing distributed agency between humans and machines.

2.	Rettberg, J.W., Kronman, L., Solberg, R. and others (2022) ‘Representations of machine vision technologies in artworks, games and narratives: A dataset’, Data in Brief, 45, 108319. https://doi.org/10.1016/j.dib.2022.108319
This data paper documents a dataset that captures cultural attitudes towards machine vision technologies as they are expressed in art, games and narratives. The dataset includes records of 500 creative works. This has given me a lot of inspiration

3.	Kronman, L. (2023) ‘Classifying Humans: The Indirect Reverse Side of Machine Vision Bias in Artworks’, Journal of Visual Culture. https://doi.org/10.1080/17540763.2023.2189160
This article examines how art exposes the classification logic and bias involved when machine vision classifies humans. Even though my work is not literally doing identity classification, it still argues that: once the body enters machine vision, it becomes an incomplete, compressed, computational version of itself.
